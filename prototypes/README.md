# Prototypes

## P2P-Exchange-v1.html

來自 Claude design 的 v1 原型(使用者端手機 + 後台桌面),由設計連結匯出之原始檔。

來源:https://claude.ai/design/p/2c7b1880-aaf0-4e7c-9bd1-496d0b4560af?file=P2P+Exchange+v1.dc.html

### 重要說明

此檔為 Claude design 的元件格式(`.dc.html`):使用自訂標籤 `<x-dc>`、`<sc-if>`、`<helmet>` 與 `{{ }}` 樣板綁定,需搭配設計執行階段 `support.js` 才能在瀏覽器中正確渲染。

- 直接以瀏覽器開啟本檔會因缺少 `./support.js` 而無法渲染。
- 完整可互動的原型請於上方 Claude design 連結檢視。

### 取得可離線開啟的單一檔(兩種方式)

1. 於 Claude design 介面使用檔案的下載/匯出功能,取得內含執行階段的單一 HTML,放到本資料夾。
2. 或由我重建一份「純 HTML + 原生 JS」的等效獨立原型(不依賴設計執行階段),可直接離線開啟。
